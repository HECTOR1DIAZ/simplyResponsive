@media queries

CSS uses the @ media rule to detect information about the device beign used to access the website. For example, it can be used to detect the size of the device, its capabilities and its orientation. For this course, we will ne using it to detect the device size, but keep in mind that it is capale of much more.

Phone: Width less than or equal to 600px

Tablet: Width between 481px and 1023px

Desktop: Width greater than or equal to 1024px
